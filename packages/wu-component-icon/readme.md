
## wu-component Icon 集合

## 使用

### install

```bash

npm install @wu-component/wu-component-icon

```

### how to use

#### 全量引入

```ts
import "@wu-component/wu-component-icon";
```

#### 单个使用

```ts
import "@wu-component/wu-component-icon";
```

```html
<wu-icon-bank-card color="red" size="28px"></wu-icon-bank-card>>

```
